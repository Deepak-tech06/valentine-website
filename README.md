# 💖 Valentine Interactive Website

An interactive Valentine website built with HTML, CSS, and JavaScript featuring animations, a coding-style intro, playful questions, gift surprises, a love quiz, photo gallery, background music, and a romantic final video.

## ✨ Features
- Coding-style intro animation
- Interactive love questions
- Moving "No" button effect
- Love quiz interaction
- Gift surprise section
- Photo gallery
- Romantic video ending
- Background music
- Floating heart animations

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
